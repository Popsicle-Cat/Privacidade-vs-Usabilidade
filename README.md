Esse é o meu TCC do curso de Sistemas de Informação na UNIRIO.

Este estudo visa contribuir com uma análise empírica que ajude a entender melhor o custo-benefício de aplicações de técnicas de privacidade diferencial em redes neurais profundas, especialmente quanto à sua viabilidade prática e impacto na performance dos modelos de aprendizagem profunda.

Busca-se entender até que ponto a privacidade diferencial pode ser efetivamente utilizada como uma ferramenta viável para proteger dados sensíveis no treinamento de redes neurais profundas, sem comprometer significativamente o desempenho ou a utilidade dos modelos treinados para a tarefa de classificação de imagens.

O artigo em PDF possui a explicação completa do trabalho realizado e dos resultados obtidos. 
Todas as imagens contidas no PDF estão também presentes nos diretórios "/graphs" e "/model_graphs" para melhor visualização das mesmas.
